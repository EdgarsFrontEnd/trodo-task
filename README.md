# Trodo Task

## Setup Instructions

```
mv .env.example .env
```

Open the .env file and update the DB variables

```
DB_DATABASE=trodo
DB_USERNAME=root
DB_PASSWORD=your_password
```

```
php artisan migrate
```

```
php artisan serve
```

```
npm install
```

```
npm run dev
```

Now you're all set! You can access your Trodo application at http://localhost:8000.

Thank you!
